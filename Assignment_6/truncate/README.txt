This C program is used to demonstrate the function of truncate to remove the data from a file at the specified offset given by the user for the given file.
input parameters:file name,Offset
input type:cmd line arguments
output type:in file
Here we take from user a file name and the offset from which the data is to br truncated and using truncate function we truncate the file,Here i have used LInux OS(Ubuntu 18.04 LTS) and performed this program on 28th Sep 2020.
