This C program is used to demonstrate stat structure which is inbuilt in UNIX OS and is used to demonstrate the stat struture and it's elements.
In this program we the stat call a file name and an empty stat structure  pointer and it return all the statistics regarding the file into that structure and we just display it.
Input type:Command Line
Input Parameter:FIle Name
Output Type:Standard Output
Prototye:stat("<file name>",&sb) where sb is empty stat structure.
