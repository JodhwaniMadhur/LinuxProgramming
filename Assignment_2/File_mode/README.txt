This program is used to check if our program which turns into process on execution has the acess to the file and if yes then which permissions i.e read,write and execute.
Input Parameter: FIle name
Input Type:Command Line Argument
Output Type:Standard output.
Prototype:access("<file name>",R_OK||W_OK||X_OK);
In this program we have used access system call to check if our process has the access to read/write/execute the file specified.
This program is built on Ubuntu Linux 18.04 LTS on September 29 2020 and is available for free usage and distribution.
