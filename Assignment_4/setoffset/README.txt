This C program is used to show lseek function.
input parameters:File Name,offset name
input:command line arguments
output:read and display 20 bytes from the specified offset
This C program specifies the lseek function working and the in this program we first open a file that is specified by the user 
and then using lseek set the offset to the specified position using SEEK_SET from the current posiotion i.e zero and set it to the specified offset and using read 
and write function display it on the screen.
This program was build and executed on Linux(ubuntu 18.04 LTS) on Sep 28 2020.This program is available for free distribution for usage and building other programs.
